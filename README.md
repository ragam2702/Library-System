# 📚 Library Management System

A desktop-based Library Management System developed using Java Swing for the graphical user interface, JDBC for database connectivity, and MySQL as the backend database.

## 🚀 Features
- ➕ Add new books to the library
- 📖 View all available books
- 📤 Issue books to users
- 🗄️ MySQL database integration
- 🖥️ Simple and interactive GUI using Java Swing

## 🛠️ Tech Stack
- Java (Swing)
- JDBC
- MySQL

## 📂 Project Structure
- `db/` → Database connection handling
- `dao/` → Data access logic (SQL operations)
- `model/` → Data models
- `ui/` → Swing user interface
- `Main.java` → Entry point

## ⚙️ Setup Instructions
1. Create the database using `database.sql`
2. Add MySQL Connector/J to the project
3. Update DB credentials in `DBConnection.java`
4. Run `Main.java`

## 🎯 Purpose
This project demonstrates CRUD operations, database connectivity, and GUI development in Java, making it suitable for academic mini-projects.
